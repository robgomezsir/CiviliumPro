Instalação

1. Extrair o ZIP

  1. Salve em C:\civilium\{{FOLDER_NAME}} (ou pasta de sua preferência).
  2. Clique com o botão direito → Extrair tudo.
  3. Confirme manifest.json, background.js e pasta icons/.

2. Carregar no Chrome

  1. Abra chrome://extensions
  2. Ative Modo do desenvolvedor
  3. Clique em Carregar sem compactação
  4. Selecione a pasta {{FOLDER_NAME}}
  5. Confirme civilium bridge versão {{VERSION}}

3. Configurar (somente Chrome Web Store)

  1. Clique no ícone da extensão → Opções
  2. URL: https://civiliumpro.vercel.app
  3. Cole a chave de integração do administrador
  4. Salvar configuração

Como usar no civilium

Confirme o badge Bridge ativa (verde) no canto superior direito do site.

Nova pesquisa

  1. Acesse https://civiliumpro.vercel.app → Nova pesquisa
  2. Cole linhas do Excel na tabela (Ctrl+V) — nome, CPF, data (DD/MM/AAAA)
  3. Ou use Buscar CSV no computador
  4. Remova linhas com o ícone de lixeira, se necessário
  5. Clique em Iniciar verificação

Na Receita Federal

  1. Resolva o CAPTCHA para cada pessoa
  2. O resultado volta ao civilium automaticamente
  3. A próxima pessoa abre na mesma aba

Atualizar

  1. Extraia o novo ZIP
  2. chrome://extensions → Recarregar
